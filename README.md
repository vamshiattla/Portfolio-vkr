## Features

- Hero section with animated subtitle and wave button
- Skills orbit showing key tools and technologies
- Sections for About, Experience, Services, Projects, Skills, and Contact
- Light / dark theme toggle

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Build the project with `npm run build` and deploy the contents of the `dist` folder to any static hosting provider (e.g., Vercel, Netlify).
